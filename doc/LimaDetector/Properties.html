Device Properties
Property name
Property type
Description
AutoStartVideo Tango::DEV_BOOLEAN Allows calling automatically the "Start" command when:
- The device starts.
- After calling the "Init" command.
AutoSaveResetRoi Tango::DEV_BOOLEAN Memorize the "full frame" automatically at the call of "ResetRoi" :
[default = false]
DetectorDescription Tango::DEV_STRING Detector user-defined text to identify the engine.
DetectorType Tango::DEV_STRING Define the type of the connected Detector .
Availables types :
- AndorCCD
- BaslerCCD
- Dhyana
- Eiger
- Hamamatsu
- ImXpad
- Lambda
- MarCCD
- Maxipix
- Merlin
- Pco
- PerkinElmer
- PilatusPixelDetector
- PrincetonCCD
- SimulatorCCD
- SlsEiger
- SlsJungfrau
- SpectralInstrument
- SpectrumOneCCD
- Ufxc
- UviewCCD
- XpadPixelDetector
DetectorPixelDepth Tango::DEV_STRING Define the pixel depth of the detector :
Availables values :
- 8
- 12
- 14
- 16
- 16S
- 24
- 24S
- 28
- 32
- 32S
SpecialDisplayType Tango::DEV_STRING Special type of the image attribute for display and saving (NOT_USED, FLOAT, ...)
DetectorVideoMode Tango::DEV_STRING Define the format of video stream:
Availables values :
- NONE
- Y8
- Y16
- Y32
- Y64
- RGB555
- RGB565
- RGB24
- RGB32
- BGR24
- BGR32
- BAYER_RG8
- BAYER_RG16
- I420
- YUV411
- YUV422
- YUV444

Nota:
This property is usefull only for detectors having video capabilities.
Otherwise, use only NONE.
ImageSource Tango::DEV_STRING Choose the source of Data given to the image attribute :
- VIDEO : use ctVideo->LastImage()
- ACQUISITION : use ctControl->ReadImage()
ImageOpMode Tango::DEV_STRING Define ImageOpMode for Roi/Binning/etc... :
Availables values :
- HardOnly
- SoftOnly
- HardAndSoft
FileFormat Tango::DEV_STRING Define the format of image files :
Availables values :
- EDF
- NXS
- RAW
FilePrefix Tango::DEV_STRING Define the prefix used for the image files name.
FileIndexPattern Tango::DEV_STRING Define the pattern of the index used for image files names.

Nota :
%04d mean that file name index will be end with exactly 4 digits filled by '0' if necessary .
FileTargetPath Tango::DEV_STRING Define the Path where Files will be generated, only when savingFile is checked.
FileNbFrames Tango::DEV_LONG Define the nombre of frames to push in each saving file.
FileManagedMode Tango::DEV_STRING Define the File managed Mode :
- HARDWARE :
- SOFTWARE :
FileWriteMode Tango::DEV_STRING Available only for Nexus format : set the SetWriteMode().
Available values :
- ASYNCHRONOUS
- SYNCHRONOUS
- DELAYED
FileMemoryMode Tango::DEV_STRING Available only for Nexus format : set the SetDataItemMemoryMode().
Available values :
- COPY
- NO_COPY
FileTimestampEnabled Tango::DEV_BOOLEAN Define wether the timestamp is requested in the Nexus file or not
DebugModules Array of string Define modules that we need to have some debug traces.
Availables values :
- None
- Hardware
- Control
- Common
- Camera
DebugLevels Array of string Define Lima verbose level of traces.
Availables values :
- Fatal
- Error
- Warning
- Trace
- Funct
- Param
- Return
- Always
DebugFormats Array of string Define Lima traces format.
Availables values :
- DateTime
- Thread
- Module
- Obj
- Funct
- FileLine
ExpertBufferMaxMemoryPercent Tango::DEV_USHORT Define the Percent of Memory reserved to Lima buffer control.
BufferMaxMemoryPercent = 70, allow a Memory of 1.4 Go. (Default)
BufferMaxMemoryPercent = 100, allow a Memory of 2 Go. (Maximum)
ExpertNbPoolThread Tango::DEV_USHORT Defines the number of threads dedicated to process images in the ProcessLib
ExpertUsePrepareCmd Tango::DEV_BOOLEAN If enabled, a call of 'Prepare' command is necessary before each 'Start' command.
ExpertTimeoutCmd Tango::DEV_ULONG Define the Timeout (in ms) for some commands (snap/start/stop/prepare).
MemorizedRoi Array of short Memorize/Define the Region of Interest of the Acquisition:
origin X
origin Y
Width
Height

Nota:
if any roi value is <0, then we consider all detector area as Roi.
MemorizedBinningH Tango::DEV_USHORT Memorize/Define the Binning Horizontal value of the Acquisition
MemorizedBinningV Tango::DEV_USHORT Memorize/Define the Binning Vertical value of the Acquisition
MemorizedAcquisitionMode Tango::DEV_STRING Memorize/Define the acquisitionMode attribute at Init device
Availables values :
- SINGLE
- ACCUMULATION
MemorizedTriggerMode Tango::DEV_STRING Memorize/Define the triggerMode attribute at Init device
Availables values :
- INTERNAL_SINGLE
- EXTERNAL_SINGLE
- EXTERNAL_MULTI
- EXTERNAL_GATE
- INTERNAL_MULTI
- EXTERNAL_START_STOP
- EXTERNAL_READOUT
MemorizedShutterMode Tango::DEV_STRING Memorize/Define the shutterMode attribute at Init device
Availables values :
- MANUAL
- AUTO_FRAME
- AUTO_SEQUENCE
MemorizedShutterOpenTime Tango::DEV_DOUBLE Memorize/Define the shutterOpenTime attribute at Init device
MemorizedShutterCloseTime Tango::DEV_DOUBLE Memorize/Define the shutterCloseTime attribute at Init device
MemorizedExposureTime Tango::DEV_DOUBLE Memorize/Define the exposureTime attribute at Init device
MemorizedExposureAccTime Tango::DEV_DOUBLE Memorize/Define the exposureAccTime attribute at Init device
MemorizedLatencyTime Tango::DEV_DOUBLE Memorize/Define the latencyTime attribute at Init device
MemorizedNbFrames Tango::DEV_LONG Memorize/Define the nbFrames attribute at Init device
MemorizedFileGeneration Tango::DEV_BOOLEAN Memorize/Define the fileGeneration attribute at Init device
MemorizedFileNbFrames Tango::DEV_LONG
SpoolID Tango::DEV_STRING
