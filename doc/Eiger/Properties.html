Eiger Class

Device Properties
Property name
Property type
Description
DetectorIP Tango::DEV_STRING Eiger server IP address (ex: 192.168.0.1)
TimestampType Tango::DEV_STRING Define the timestamp type of each frame:
- RELATIVE : The time difference between the moment of reading the frame and the start acquisition time
- ABSOLUTE : The time at the moment of reading the frame from detector
CurlDelayMs Tango::DEV_DOUBLE Curl delay in ms. this is used as a sleep delay for waiting the curl responses
DownloadDataFile Tango::DEV_BOOLEAN Enable/Disable downloading data files from DCU. Do not download data files (master+data) [by default]
NbFramesPerTriggerIsMaster Tango::DEV_BOOLEAN If True: nbFrames = memorized values of NbTriggers * NbFramesPerTrigger (case on PX1 beamline) If False: nbFrames = memorized nbFrames (case on Swing, Sixs beamlines)
MemorizedCountrateCorrection Tango::DEV_BOOLEAN Memorize the value of countrateCorrection attribute.
MemorizedFlatfieldCorrection Tango::DEV_BOOLEAN Memorize the value of flatfieldCorrection attribute.
MemorizedPixelMask Tango::DEV_BOOLEAN Memorize the value of pixelMask attribute.
MemorizedVirtualPixelCorrection Tango::DEV_BOOLEAN Memorize the value of virtualPixelCorrection attribute.
MemorizedPhotonEnergy Tango::DEV_DOUBLE Memorize the value of photonEnergy attribute.
MemorizedAutoSummation Tango::DEV_BOOLEAN Memorize the value of autoSummation attribute.
MemorizedCompression Tango::DEV_BOOLEAN Memorize the value of compression attribute.
MemorizedCompressionType Tango::DEV_STRING Memorize the value of compressionType attribute.
MemorizedRoiMode Tango::DEV_STRING Memorize the value of roiMode attribute.
MemorizedBeamCenterX Tango::DEV_DOUBLE Memorize the value of beamCenterX attribute.
MemorizedBeamCenterY Tango::DEV_DOUBLE Memorize the value of beamCenterY attribute.
MemorizedDetectorDistance Tango::DEV_DOUBLE Memorize the value of detectorDistance attribute.
MemorizedChiIncrement Tango::DEV_DOUBLE Memorize the value of chiIncrement attribute.
MemorizedChiStart Tango::DEV_DOUBLE Memorize the value of chiStart attribute.
MemorizedKappaIncrement Tango::DEV_DOUBLE Memorize the value of kappaIncrement attribute.
MemorizedKappaStart Tango::DEV_DOUBLE Memorize the value of kappaStart attribute.
MemorizedOmegaIncrement Tango::DEV_DOUBLE Memorize the value of omegaIncrement attribute.
MemorizedOmegaStart Tango::DEV_DOUBLE Memorize the value of omegaStart attribute.
MemorizedPhiIncrement Tango::DEV_DOUBLE Memorize the value of phiIncrement attribute.
MemorizedPhiStart Tango::DEV_DOUBLE Memorize the value of phiStart attribute.
MemorizedNbTriggers Tango::DEV_LONG Memorize the value of nbTriggers attribute.
MemorizedNbFramesPerTrigger Tango::DEV_LONG Memorize the value of nbFramesPerTrigger attribute.
